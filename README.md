# Restaurant Channel Profitability Analysis

## Overview
This project analyzes profitability across multiple restaurant sales channels:
- In-Store Dining
- Uber Eats
- DoorDash
- Self Delivery

The analysis focuses on understanding cost structures, margin efficiency, and channel-wise performance using a Streamlit dashboard.

## Objectives
- Compare net profitability across channels
- Analyze margin erosion due to commissions and delivery costs
- Identify cost-efficient channels
- Evaluate profitability across cuisine types and business segments

## Features
- Channel-wise profit comparison
- Margin efficiency analysis
- Cost structure breakdown
- Cuisine and segment profitability heatmaps

## Technologies Used
- Python
- Pandas
- Streamlit
- Plotly
- Seaborn
- Matplotlib

## How to Run
1. Clone the repository
2. Install dependencies:
   pip install streamlit pandas matplotlib seaborn plotly
3. Run the app:
   streamlit run app.py

## Dataset
The dataset contains Indian restaurant data including:
- Order distribution across channels
- Revenue and cost parameters
- Profitability metrics
